Unordered HTML List: tarts with the "ul" tag. Each list item starts with the "li" tag.

The list items will be marked with bullets (small black circles) by default

  • Coffee
  • Tea
  • Milk

Unordered HTML List - Choose List Item Marker - disc - circle - square - none

"ul style="list-style-type:circle"

  • Coffee
  • Tea
  • Milk

"ul style="list-style-type:none"

  • Coffee
  • Tea
  • Milk

Ordered HTML List: starts with the "ol" tag. Each list item starts with the "li" tag.

  1. Coffee
  2. Tea
  3. Milk

Ordered HTML List - The Type Attribute - "1" - "A" - "a" - "I" - "i"

"ol type="1""

  1. Coffee
  2. Tea
  3. Milk

"ol type="I""

  1. Coffee
  2. Tea
  3. Milk

HTML Description Lists

The "dl" tag defines the description list,

the "dt" tag defines the term (name),

and the "dd" tag describes each term:

Coffee
- black hot drink
Milk
- white cold drink
