HTML Iframes

An iframe is used to display a web page within a web page

Syntax: <iframe src="URL"></iframe>

Set Height and Width: height="200" width="300"

Iframe - Remove the Border: By default, an iframe has a border around it: style="border:none;"

Target for a Link: The target attribute of the link must refer to the name attribute of the iframe

MasBytes

When the target of a link matches the name of an iframe, the link will open in the iframe.
