HTML Form Elements

The <input> Element can be displayed in several ways, depending on the type attribute.

If s omitted, the input field gets the default type: "text".

<input name="firstname" type="text">

The <select> Element, defines a drop-down list:

The <option> elements defines an option that can be selected.
By default, the first item in the drop-down list is selected.
To define a pre-selected option, add the selected attribute to the option:

<option value="fiat" selected>

Visible Values:

to specify the number of visible values: size="3"

Allow Multiple Selections:

Use the multiple attribute to allow the user to select more than one value: multiple

The <textarea> Element, defines a multi-line input field

The rows attribute specifies the visible number of lines in a text area.
The cols attribute specifies the visible width of a text area: rows="10" cols="30"

You can also define the size of the text area by using CSS: style="width:200px; height:600px"

The <button> Element: type="button" onclick="alert('Hello World!')"
