HTML File Paths

File Paths: A file path describes the location of a file in a web site's folder structure.

<img src="picture.jpg"> - file is located in the same folder as the current page.
<img src="images/picture.jpg"> - file is located in the images folder in the current folder
<img src="/images/picture.jpg"> - file is located in the images folder at the root of the current web
<img src="../picture.jpg"> - file is located in the folder one level up from the current folder

Relative File Paths points to a file relative to the current page.

file is in the images folder located at the root of the current web: <img src="/images/picture.jpg">

file in the images folder located in the current folder: <img src="images/picture.jpg">

a file in the images folder located in the folder one level above the current folder: <img src="../images/picture.jpg">

Best Practice

When using relative file paths, your web pages will not be bound to your current base URL. All links will work on your own computer (localhost) as well as on your current public domain and your future public domains.
