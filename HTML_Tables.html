HTML Tables

Defining an HTML Table

Is defined with the "table" tag.
Each table row is defined with the "tr" tag.
A table header is defined with the "th" tag.
By default, table headings are bold and centered.
A table data/cell is defined with the "td" tag.

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94

HTML Table - Adding a Border: "border: 1px solid black;"

HTML Table - Collapsed Borders: "border-collapse: collapse;"

HTML Table - Adding Cell Padding: The space between the cell content and its borders: "padding: 15px;"

HTML Table - Left-align Headings: table headings are bold and centered By default: "text-align: left;"

HTML Table - Adding Border Spacing: specifies the space between the cells: "border-spacing: 5px;"

HTML Table - Cells that Span Many Columns: To make a cell span more than one column,
use the colspan attribute: "th colspan="2""

HTML Table - Cells that Span Many Rows: To make a cell span more than one row,
use the rowspan attribute: "th rowspan="2""
